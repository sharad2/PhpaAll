Overview

This document discusses how to use MoneyField for overriden feature. this feature is usefull for money value which have some default value and later user can override this amount. if user changed default value then an image is dispalyed in read ony mode to indicate an default value has been overridden. in the case of edit a check box is dispalyed. In edit mode a checked checkbox is shown. If OverrideDataField evaluates to false, the checkbox is not shown in edit mode and info image is not shown in read mode.
  1. • Add MoneyField in gridView.
  2. • set OverrideDataField property to DataField which decide its value is overriden or not.
  3. •Set OverrideToolTip which is the tooltip of image
  4. • Now add GridView and assign datasource created above to this gridview. Framework asks you to regenerate gridview column fields and data keys when you press finish button, do not regenerate them.Refresh schema.
  5. Make sure that the DataKeyName of the grid has been set to the primary key of the table you are selcting from.
  6. • Set the default sort column of the grid and make sure that sorting is working.
  7. Enable paging if required.
  8. • If you want to have a custom query then handle the Selecting event of the datasource.You must select all the columns that the datagrid is showing.
