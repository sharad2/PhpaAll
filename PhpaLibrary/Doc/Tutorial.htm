Voucher Entry Steps:
  1. The user must be a logged in user since he is the only one who can give some input for data entry.
  2. The user must insert all the heads which are required for entering any specific voucher.
  3. Voucher Entry
Details of the voucher entered can bee seen in
  1. Day Book
  2. Ledger
  3. Journal (Will display all the vouchers other than cash & bank)
Cash, Bank Report : For viewing specific reports on cash or bank
  1. Define head type for each head respectively, Account Head will provide this facility
  2. All the above defined reports can be viewed which provide a much better segrerated information of cash or bank
The organization may have received some cash from contractor as Earnest Money Deposit or Security Deposit for getting those details
  1. Trace the list if contractor exist, if not create a new one for the same
  2. Provide the contractor detail against the voucher, these details can be provided either at the time when new voucher is created or later in edit mode
Paying advances to employees is a common scenario so there must be some report by which details of balance employee advances could be known, for getting these details
  1. The Employee must exist, if this is not the case provide the entry for the same
  2. Define the head type for Employee Advances
  3. Provide employee information for the voucher of employee advances
Expenditure Reports
  1. Provide the account type against the head for which expenditure reports to be viewed as Expenditure though Account Head
Job Expenditure Report, Balance Project Cost & Expenditure under various job types
  1. Create a division for which the job is to be created
  2. Create a Job for the task to be performed, in job do specefy for which head the job is being created
  3. Provide Job information for the voucher which comes under this job
  4. One of the head involved in the voucher must be of type expenditure
Detailed reports
  1. Receipts & Payments
  2. Trial Balance
  3. Balance Sheet
