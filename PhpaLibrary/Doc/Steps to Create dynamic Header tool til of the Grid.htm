Overview

This document discuss, how to create Dynamic Header Tool tip in the grid with respect to the Date parameter passed.
  1. Code is best worked in Row created event of the grid.
  2. First Create a DateTime variable using the date parameters passed.
  3. Get the value of the Grid column index using FindByAccessibleHeaderText in Int type variable i.
  4. Cast your grid column having index i in corresponding grid field. e.g.MoneyField and MultiBound field etc.
  5. Now, you can access all the properties of the grid column having index i.
  6. Set the HeaderToolTip of the grid column using string.Format and passed date variables accessed from the Enhanced Text box having DataType as Date.
  7. Refer gvExpenditure_RowCreated event in Expenditure.cs report.
